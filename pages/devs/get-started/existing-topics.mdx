## Overview

**Some useful topics have already been created.**

**The Allora Appchain already contains the following topics on Testnet.** Below, you'll find the topic ID, name, and a brief description for each.

### Why Explore Existing Topics?

Existing topics provide:
- **Learning opportunities**: Understand how topics are structured and function
- **Participation examples**: See real-world prediction scenarios in action
- **Development references**: Use as templates for your own topic creation
- **Network activity**: Observe active predictions and participant behavior

### Topic Categories

The current topics span several categories:
- **Cryptocurrency prices**: Short-term and medium-term predictions for major assets
- **Volatility predictions**: Risk and volatility forecasting for crypto assets
- **Political events**: Election outcome predictions
- **Memecoin tracking**: Predictions for emerging token markets

## Important Considerations

<Callout type="warning">
**Warning**: Topic ordering is never guaranteed to be consistent between separate chains/deployments.
</Callout>

**Key Implications**:
- Topic IDs may differ between testnet and mainnet
- Always verify topic details before building applications
- Use topic metadata rather than IDs for application logic
- Test thoroughly when moving between different network environments

### Development Best Practices

**When Using Existing Topics**:
- Query topic details programmatically rather than hardcoding IDs
- Build flexible systems that can adapt to topic changes
- Monitor topic activity and participation patterns
- Use existing topics as learning examples before creating new ones

## Getting Started with Topics

### Prerequisites

Before interacting with these topics:
- [Install `allorad`](/devs/get-started/cli#installing-allorad) for command-line interactions
- Set up your wallet and fund it from the testnet faucet
- Understand basic blockchain query operations
- Review topic creation documentation for background knowledge

### Next Steps with Existing Topics

**Exploration Activities**:
1. Query existing topic data to understand their structure
2. Monitor inference submissions and results
3. Analyze participant behavior and performance patterns
4. Study different prediction methodologies in use

**Participation Options**:
1. Deploy workers to submit predictions on existing topics
2. Delegate stake to reputers evaluating these topics
3. Consume inference data for your applications
4. Study network dynamics through active topics

## Creating Your Own Topics

Once you understand existing topics, you can [create your first topic](/devs/topic-creators/how-to-create-topic) by following the instructions in the hyperlinks provided.

**When to Create New Topics**:
- You have unique prediction requirements not covered by existing topics
- You want to experiment with different time horizons or methodologies
- You're building applications requiring specialized inference data
- You want to contribute new prediction categories to the network
