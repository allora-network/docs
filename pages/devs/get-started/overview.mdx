## Overview

**Welcome to the Allora developer's guide!** This page will help you get up to speed with the Allora repositories that will be used throughout our documentation and show you how to start contributing to our ecosystem.

### What is Allora?

**Allora is a decentralized network that leverages the power of collective participation in tasks such as data inference, forecasting, and verification.** [Contribute](/community/contribute) to any of the repositories below to help enhance and grow the Allora ecosystem.

### Why Choose Allora?

**Benefits of the Allora ecosystem**:
- **Decentralized intelligence**: Harness collective knowledge for better predictions
- **Incentivized participation**: Earn rewards for valuable contributions
- **Open collaboration**: Contribute to cutting-edge decentralized AI research
- **Flexible roles**: Choose from multiple participation paths based on your expertise

## Core Infrastructure

### Allora Chain

**The [Allora chain](https://github.com/allora-network/allora-chain) is a Cosmos Hub chain that forms the core of the Allora network.** This repository contains the blockchain's codebase and is useful for validators.

**Key Features**:
- **Cosmos SDK foundation**: Built on proven blockchain technology
- **Validator-friendly**: Comprehensive tools for network validation
- **Transaction processing**: Handles all network operations and rewards

**You can use the [`allorad` CLI tool](/devs/get-started/cli#installing-allorad) to query the chain and make transactions.**

<Callout>
Some subsections in the Table of Contents below require downloading and installing [`allorad`](/devs/get-started/cli#installing-allorad).

As a general rule, it is suggested to download and install `allorad` for any developer looking to interact with the network.
</Callout>

**CLI Benefits**:
- **Network interaction**: Essential tool for all network operations
- **Data querying**: Access comprehensive network information
- **Transaction management**: Submit and manage blockchain transactions

## Getting Started Recommendations

### For New Developers

**Recommended Learning Path**:
1. **Start with [Setup Wallet](/devs/get-started/setup-wallet.mdx)** - Essential foundation for all network interactions
2. **Explore [Basic Usage](/devs/get-started/basic-usage.mdx)** - Learn fundamental concepts and operations
3. **Review [Existing Topics](/devs/get-started/existing-topics.mdx)** - Understand current network opportunities
4. **Choose your role** - Select from Workers, Reputers, Validators, or Consumers based on your interests and expertise

### For Experienced Blockchain Developers

**Advanced Path**:
1. **Quick setup** with [Setup Wallet](/devs/get-started/setup-wallet.mdx) and [Basic Usage](/devs/get-started/basic-usage.mdx)
2. **Deep dive** into your chosen role's documentation section
3. **Explore integration options** through Consumer or Validator documentation
4. **Contribute** to the ecosystem through [community channels](/community/contribute)

### For AI/ML Practitioners

**ML-Focused Path**:
1. **Essential setup** through wallet and basic usage guides
2. **Focus on [Workers](/devs/workers/workers.mdx)** section for model deployment
3. **Study [Reputers](/devs/reputers/reputers.mdx)** for quality assessment and evaluation
4. **Join** the Model Forge Competition for hands-on experience
