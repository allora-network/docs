# How to get faucet funds

Obtain some funds to participate in the Allora Edgnet/Testnet by accessing https://faucet.edgenet.allora.network or https://faucet.testnet.allora.network/ for the Edgenet or Testnet faucets, respectively.

Enter the Allora Wallet address for the account that needs funding. If you don't have a wallet created yet, use the instructions [here](../learn/cli#create-a-wallet) to create one.
