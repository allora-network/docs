# Overview

Welcome to Allora! This Quickstart will help you understand what Allora is and get you started with the platform.

## What is Allora?

The Allora Network is a state-of-the-art protocol that uses decentralized AI and machine learning (ML) to build and deploy predictions among its participants. It offers actors who wish to use AI predictions a formalized way to obtain the output of state-of-the-art ML models on-chain and to pay the operators of AI/ML nodes who create these predictions.

## Key Features

- **Decentralized AI Network**: Leverage collective intelligence for predictions
- **On-chain ML Models**: Access state-of-the-art machine learning predictions
- **Incentive Mechanism**: Rewards for high-quality predictions and assessments
- **Continuous Learning**: Network adapts and improves over time

## 📚 Key Terminology

New to Allora and need help understanding the terminology? Check out our **[Key Terms Guide](/home/key-terms)** for clear definitions of:

- **Topics** - Prediction markets and inference coordination
- **Workers** - AI/ML nodes providing predictions
- **Reputers** - Validators assessing prediction accuracy
- **Inference** - AI-generated predictions and outputs
- And much more!
