## What is Model Forge?

The [Allora Model Forge Competition](https://forge.allora.network) is an open-source hackathon where participants:

- Create AI/ML models for specific prediction topics
- Compete against other developers
- Earn rewards based on model performance
- Contribute to the Allora ecosystem

## Competition Features

- **Live Leaderboards** - Real-time ranking of model performance
- **Multiple Topics** - Various prediction challenges to choose from
- **Fair Scoring** - Transparent evaluation using Mean Squared Error
- **Rewards** - Earn tokens for high-performing models

## Current Competition Topics

1. **[5min ETH Price Prediction](https://forge.allora.network/competitions/1)**
2. **[12hr USDC/ETH Volume Prediction](https://forge.allora.network/competitions/2)**

## How to Participate

### Prerequisites

Before joining, make sure you have:
- **[Allora CLI installed](/getting-started/install-cli)**
- **[Wallet created](/getting-started/create-wallet)**
- **[Testnet tokens](/getting-started/get-testnet-tokens)** for registration
- **Basic AI/ML knowledge**

### Registration Process

1. **Sign Up** - Register at the [Sign Up Page](https://vk4z45e3hne.typeform.com/to/ypA2Yl1J?utm_source=landing-page&typeform-source=forge.allora.network)
2. **Wait for Acceptance** - You'll be notified when accepted
3. **Setup Environment** - Prepare your development environment
4. **Deploy Worker** - Create and deploy your predictive model
5. **Monitor Performance** - Track your model on the leaderboard

## What You'll Learn

Participating in Model Forge teaches you:

- **Allora Network Concepts** - Topics, workers, reputers, and epochs
- **Model Development** - Creating effective predictive models
- **Network Interaction** - Using the Allora CLI and APIs
- **Performance Optimization** - Improving model accuracy over time

## Competition Rules

- **Loss Function**: Mean Squared Error (MSE)
- **Epoch Timing**: Varies by topic (5min, 1hr, 1d)
- **Scoring**: Based on prediction accuracy
- **Fair Play**: Open-source and transparent evaluation
